1) Navigation
cd <folder_name> 
--> goes into the folder 

cd ..
--> goes up one level

cd \
--> go to root 

dir 
--> list files/folders

dir /a files 
--> includes hidden files

2) File and Folder Management
mkdir folder
--> create a folder

rmdir folder
--> delete empty folder

rmdir /s folder 
--> delete folder + contents

type nul > a.txt 
--> create empty file

del file.txt 
--> delete file
